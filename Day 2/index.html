VS Code Shortcuts

  • Ctrl + Shift + P → Command Palette
  • Ctrl + B → Hide / Unhide Explorer
  • Ctrl + , → Open Settings

Extensions

  • Material Icons Theme → cool icons for files and folders
  • Language IntelliSense & Snippets → Code suggestions and snippets
  • Prettier → Used to format code, but VS Code now has built-in formatting (based on Prettier). May not be necessary to install separately to avoid bloating the editor.
  • Better Comments → Highlight comments using !, TODO, *, ?
  • Quokka.js → Real-time outputs and errors while writing JavaScript/TypeScript
  • Live Server → Run and preview code locally
  • Auto Rename Tag → Automatically updates closing tags when editing opening tags

Tips

  • element.classname → Quickly create an element with a class
  • element#id → Quickly create an element with an ID

CSS

Basic CSS syntax:

selector {
    property: value;
  }

Flexbox

Flexbox is a layout model that allows easy alignment and distribution of space among items in a container.

display: flex;
justify-content: center | space-between | space-around;
align-items: center | flex-start | flex-end;

Example:

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

CSS Grid

CSS Grid is used for creating 2-dimensional layouts using rows and columns.

display: grid;
grid-template-columns: repeat(3, 1fr);
grid-gap: 20px;

Example:

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 20px;
}
